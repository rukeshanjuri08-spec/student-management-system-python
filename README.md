# student-management-system-python

## Description
A command-line based Student Management System built using Python. 
It allows users to manage student records with persistent storage.

## Features
- Add, View, Search, Delete, Update students
- Data stored using JSON file
- Input validation for accuracy
- Case-insensitive search
- Sorting functionality
- Count total students

## Technologies Used
- Python
- JSON

## How to Run
1. Download the code
2. Run the Python file
3. Use menu options to manage students
